CAL 모바일 리포트 생성기

마스터 JSON을 붙여넣고, 개인 전용 모바일 HTML 리포트를 생성합니다.

1. JSON 입력

기존 마스터 JSON을 그대로 붙여넣으세요. JSON 객체만 있어도 되고, const CAL_REPORT_DATA = {...} 형태여도 됩니다.

대기 중 · JSON을 입력한 뒤 미리보기 또는 다운로드를 누르세요.

권장 전달 흐름: 생성된 HTML 파일 → zip 압축 → 카톡 전달 → Chrome 또는 HTML 뷰어로 열기.

2. 모바일 미리보기

실제 고객 휴대폰에서 보일 폭에 가깝게 390px 기준으로 표시합니다.
